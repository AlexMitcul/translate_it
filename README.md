# Translate IT -> Translation Application

## Requirements
- Kotlin
- Maven
- Google Translate API Key

## Running the Application
1. Clone the repository
    ```bash
    git clone https://github.com/AlexMitcul/translate_it.git
    cd translate_it
    ```
2. Build the project
    ```bash
    mvn clean install
    ```
3. Run the application
    ```bash
    mvn spring-boot:run
    ```
